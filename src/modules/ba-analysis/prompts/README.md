# Agent Prompts

System prompts for each agent (Alex, Emma, Sarah, David, Paul). Follow the structure in repo root `master_prompt.txt`:

- `alex.prompt.txt` – Senior BA orchestrator
- `emma.prompt.txt` – Requirements validation
- `sarah.prompt.txt` – Stakeholder mapping
- `david.prompt.txt` – BABOK compliance
- `paul.prompt.txt` – Traceability and missing links

Each prompt should include: `<persona>`, `<input_specification>`, `<core_directives_and_principles>`, `<output_blueprint>` (and optionally `<internal_pre_interaction_meta_cot>`, `<interaction_protocol>`).
